# Foundation Block Calculator

A stepped footing block calculator for masonry construction.

## Deploy

This repo is set up to deploy on [Railway](https://railway.com) using Nginx.

1. Push this repo to GitHub
2. Go to railway.com → New Project → Deploy from GitHub repo
3. Select this repo
4. Railway auto-detects the Dockerfile and deploys
5. Go to Settings → Networking → Generate Domain to get your public URL
